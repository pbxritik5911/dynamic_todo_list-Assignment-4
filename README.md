# ✨ Dynamic To-Do List Web App

A modern and responsive To-Do List application built using **HTML, CSS, and JavaScript**. This project allows users to efficiently manage daily tasks with a clean UI and dynamic functionality.

---

## 🚀 Features

- ➕ Add new tasks  
- ✏️ Edit existing tasks  
- ❌ Delete tasks  
- ✅ Mark tasks as completed  
- 📊 Real-time task statistics (Total / Completed / Pending)  
- 💾 LocalStorage support (data saved even after refresh)  
- 🎨 Modern UI with responsive design  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- LocalStorage API
